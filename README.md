Never Lose
==========

This Chrome extension save all web pages you viewed to the Wayback Machine

Domains with subdomains excluded:
* amazon.com
* apple.com
* archive.org
* bing.com
* evart.com
* facebook.com
* gmail.com
* google.com
* vk.com
* wikipedia.org
* yahoo.com
* yandex.ru

And any IP address
