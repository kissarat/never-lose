Never Lose
==========

This Chrome extension save all web pages you viewed to the Wayback Machine
[Download](https://github.com/kissarat/never-lose/raw/master/build/never-lose.crx)

Domains with subdomains excluded:
* amazon.com
* apple.com
* archive.org
* bing.com
* evart.com
* facebook.com
* gmail.com
* google.com
* vk.com
* wikipedia.org
* yahoo.com
* yandex.ru

And any IP address
